# Ecoleta 

Site de cadastro e consulta de locais de coleta de lixos recicláveis

## Installation

Pós node.js instalado, basta digitar no terminal: 
```
npm install
```

## Usage

Ainda no terminal dê o start e acesse no navegador em localhost/3000
```
npm start
```
